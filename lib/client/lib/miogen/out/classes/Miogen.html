Parameters:

  • who Object

    The owner of this bind which can later be used to unbind.

  • obj Object

    The target object which is being bound to.

  • method String

    The name of the method that will be bound to.

  • cb Function

    The callback method to call when the binded method has been executed. The arguments of this callback will be the same of the method being bound.

Defined in js/Miogen.js:95

Standardised method to record application errors with the option to notify the user.

Parameters:

  • message String | Exception

    The error message.

  • notifyUser Boolean

    Optional default false. When true, the user will be notified of the error.

Example:

    doSomething();
}
catch (ex) {
    Miogen.logError(ex);
}
